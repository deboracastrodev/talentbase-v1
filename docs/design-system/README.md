## 🚀 Quick Start

### Cores Principais
```css
--color-primary: #00B8D4;
--color-background-dark: #000000;
--color-background-light: #FFFFFF;
```

### Tipografia
```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

## 📖 Como Usar
