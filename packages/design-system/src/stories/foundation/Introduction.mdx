# 🎨 TalentBase Design System

> Sistema de design oficial da **TalentBase** - Sua plataforma de recolocação profissional com tecnologia e humanização.

## Bem-vindo

Este design system foi criado para garantir **consistência, qualidade e eficiência** em todos os produtos TalentBase. Aqui você encontrará todos os componentes, padrões e diretrizes necessários para criar experiências excepcionais.

---

## 📖 O que você encontrará

### **Foundation (Fundamentos)**
Os elementos base que formam nossa identidade visual:
- **Brand** - Logo, cores e identidade da marca
- **Colors** - Paleta de cores e tokens
- **Typography** - Tipografia e hierarquia
- **Spacing** - Sistema de espaçamentos
- **Icons** - Biblioteca de ícones

### **Components (Componentes)**
Componentes reutilizáveis prontos para uso:
- Buttons, Cards, Forms
- Navigation, Modals, Badges
- Layouts e Templates

### **Patterns (Padrões)**
Padrões de composição e boas práticas.

---

## 🚀 Como Usar

### 1. Instalação

```bash
npm install @talentbase/design-system
```

### 2. Importar Componentes

```tsx
import { Button, Card } from '@talentbase/design-system';

function App() {
  return (
    <Card>
      <Button variant="primary">Clique aqui</Button>
    </Card>
  );
}
```

### 3. Configurar Tailwind

```js
// tailwind.config.js
module.exports = {
  presets: [
    require('@talentbase/design-system/tailwind.config')
  ],
  // ... suas configurações
}
```

## 💡 Princípios de Design

### **1. Humanização**
Criamos experiências que respeitam e valorizam as pessoas, com linguagem acessível e empática.

### **2. Simplicidade**
Interfaces claras e diretas, removendo complexidade desnecessária.

### **3. Consistência**
Padrões visuais e de interação previsíveis em toda a plataforma.

### **4. Acessibilidade**
Design inclusivo que funciona para todos, seguindo WCAG 2.1 AA.

### **5. Performance**
Componentes otimizados para carregamento rápido e experiência fluida.
